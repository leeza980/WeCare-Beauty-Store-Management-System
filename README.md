

# WeCare Beauty Store Management System

A Python-based inventory management system designed for managing beauty store products.

## Features

- Product Inventory Management
- Product Sales Processing
- Product Restocking
- Invoice Generation
- Customer Management
- Supplier Management
- Input Validation
- File-Based Data Storage

## Technologies Used

- Python
- File Handling
- Modular Programming

## Project Structure

```text
main.py          # Main menu and system flow
operations.py    # Business logic
display.py       # Display and invoice functions
read.py          # Read data from file
write.py         # Save data to file
products.txt     # Product database
```
